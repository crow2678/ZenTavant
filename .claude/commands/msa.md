Use the tavant-universe MCP server to create a Master Services Agreement (Professional Services Agreement).

The MSA uses Tavant's standard legal language — do NOT modify the legal content.

Create the MSA using:
1. msa_create — initialize with customer name and effective date
2. msa_add_section — add ALL 13 sections in order: cover_page, preamble, definitions, professional_services, acceptance_and_fees, ownership, confidentiality, warranties, indemnification, limitation_of_liability, term_and_termination, general_provisions, signatures
3. msa_export — save the .docx file

Customer details: $ARGUMENTS
