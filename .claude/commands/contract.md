Use the tavant-universe MCP server to create a Contract (Services Agreement).

Create the contract using:
1. contract_create — initialize with contract title, client name, and effective date
2. contract_add_section — add sections in order: cover_page, parties, scope_of_work, timeline, commercial_terms, confidentiality, ip_rights, termination, liability, general_provisions, signatures
3. contract_export — save the .docx file

Contract details: $ARGUMENTS
