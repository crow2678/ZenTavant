Use the tavant-universe MCP server to create a Mutual Non-Disclosure Agreement.

The NDA uses Tavant's standard legal language — do NOT modify the legal content.

Create the NDA using:
1. nda_create — initialize with company name and effective date
2. nda_add_section — add ALL 11 sections in order: cover_page, preamble, proprietary_information, protection, exclusions, rights, legends, general_terms, term, entire_agreement, signatures
3. nda_export — save the .docx file

Company details: $ARGUMENTS
