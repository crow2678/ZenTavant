Use the tavant-universe MCP server to create a Statement of Work.

Create the SOW using:
1. sow_create — initialize with project title, client name, and date
2. sow_add_section — add sections in order: cover_page, overview, scope, approach, deliverables, timeline, team, pricing, assumptions, governance, acceptance, signatures
3. sow_export — save the .docx file

Project details: $ARGUMENTS
