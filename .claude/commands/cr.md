Use the tavant-universe MCP server to create a Change Request document.

Create the CR using:
1. cr_create — initialize with customer name, project name, and CO number
2. cr_add_section — add sections in order: cover_page, background, project_details, charges, invoicing, sow_reference, counterparts, signatures
3. cr_export — save the .docx file

Change request details: $ARGUMENTS
